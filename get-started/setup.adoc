= BlueXP移行レポートをセットアップ

:allow-uri-read: 
:icons: font
:imagesdir: ../media/get-started/


[role="lead"]
BlueXP移行レポートサービスを設定する必要はありません。BlueXP移行レポートオプションは自動的に有効になり、BlueXPの左側のナビゲーションに表示されます。
